Cyber Warrior .PROD - Director's Dashboard

Overview

Cyber Warrior .PROD is a specialized, interactive Single Page Application (SPA) designed by Cyber SoHo to function as the "Mission Control" for the production of the Red Deer Polytechnic New Cybersecurity Content and Courses' Advertisement.

This dashboard transforms the production script into an interactive director's monitor, allowing for real-time tracking of shots, logistical planning by location, and analysis of video pacing/energy.

Features

Interactive Storyboard: Step-by-step execution guide with location filtering (Gym, Server Room, Office, Greenhouse) to batch filming efficiently.

Production Analytics:

Energy Arc: Visualizes the aggressive vs. comedic pacing of the video.

Location Breakdown: Logistics planning tool.

Budget Intelligence: Visual comparison of "Classical Agency" costs vs. the "AI-Powered" production model employed in this project.

Tech Specs: Quick reference for camera settings (iPhone 16 Pro Max, Insta360) and Audio cues.

Installation & Usage

Download: Save the Production_Dashboard.html file to your local device (computer or tablet).

Run: Open the file in any modern web browser (Chrome, Safari, Edge).

Note: No internet connection is required to run the logic, but an internet connection is needed to load the Tailwind CSS and Chart.js libraries via CDN.

Deploy: (Optional) Upload to GitHub Pages or Netlify for remote access on set.
